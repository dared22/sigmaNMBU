## module.description

This workshop provides a hands-on introduction to neural networks with PyTorch. We start with fundamental concepts like perceptrons and activation functions, building up to training a simple image recognition network.

### Prerequisites

- Basic Python knowledge
- Linear algebra (matrices, vectors)
- Laptop with Python 3.10+ installed
